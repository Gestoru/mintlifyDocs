## 📊 Reporte de Ventas Generales  

El **reporte de ventas generales** permite analizar el detalle de las facturas de venta generadas, identificar tendencias comerciales y exportar la información para su gestión.

<Info>
Este reporte te ayuda a evaluar el desempeño comercial y realizar análisis estratégicos basados en datos reales.
</Info>

---

## 🔍 Cómo Interpretarlo  

### 🔹 Opciones de Filtros  

- **Cliente:** Consulta ventas de un cliente específico o todas las ventas.  
- **Vendedor:** Filtra ventas por vendedor asignado.  
- **Numeración:** Define el tipo de numeración a consultar.  
- **Rango de Fechas:** Selecciona un período específico con los campos "Desde" y "Hasta".  
- **Período Rápido:** Accede a opciones como *Hoy, Este Mes, Este Año*.  

### 🛠 Botones de Acciones  

- **Generar Reporte:** Muestra la información según los filtros aplicados.  
- **Exportar a Excel:** Descarga los datos en formato Excel.  
- **Guardar Cierre:** Guarda el reporte para análisis posterior.  
- **Facturas con Órdenes de Compra:** Relaciona facturas con órdenes de compra.  

---

## 📊 Estructura del Reporte Generado  

Este reporte se presenta en una tabla con las siguientes columnas:

| Columna      | Descripción |
|-------------|------------|
| **Nro.** | Número de la factura |
| **Cliente** | Nombre del cliente asociado |
| **Vendedor** | Asesor responsable de la venta |
| **Usuario** | Persona que generó la factura |
| **Creación** | Fecha de emisión |
| **Canal** | Medio de venta utilizado |
| **Subtotal** | Valor antes de impuestos |
| **IVA** | Impuesto aplicado |
| **Retención** | Valor retenido según normativa |
| **Nota C.** | Indica si hay una nota crédito asociada |
| **Pagado** | Monto pagado por el cliente |

<Check>
En la parte inferior del reporte, se muestra un resumen totalizado de las ventas, incluyendo el subtotal y el IVA.
</Check>
