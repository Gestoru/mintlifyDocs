## 1. Asociación de la Resolución en Gestoru

### Descripción breve
Este módulo explica el proceso para asociar la resolución de facturación en Gestoru. Al seguir estos pasos, se asegura la correcta integración con la DIAN para la emisión de facturas a través del sistema POS.

### Características principales
- *Acceso a Configuración:* Navega al menú principal y selecciona la sección de configuración.
- *Gestión de Numeraciones:* Acceso a la opción de numeraciones para gestionar las resoluciones.
- *Creación de Numeración:* Uso del botón “Nueva Numeración” para iniciar el proceso de asociación.
- *Soporte Visual:* Se recomienda usar capturas de pantalla y videos para complementar el proceso.

<Card title="Acceso Rápido">
  Dirígete a *Configuración* y luego a *Facturación POS* para comenzar la asociación de la resolución.
</Card>

### Pasos para asociar la resolución
<Steps>
  <Step title="Acceder a Configuración">
    Ingresa al menú principal y selecciona *Configuración*.
  </Step>
  <Step title="Entrar a Facturación POS">
    Dentro de Configuración, haz clic en *Numeraciones*.
  </Step>
  <Step title="Crear Nueva Numeración">
    Selecciona el botón *“Nueva Numeración”* para iniciar el proceso.
  </Step>
</Steps>

<Note>
  Se recomienda incluir una imagen del formulario “Nueva Numeración” para mayor claridad visual.
</Note>

### Recomendaciones generales
- Verifica que accedes a la sección correcta del sistema.
- Comprueba que la interfaz coincide con las indicaciones del manual.
- Consulta los recursos adicionales para más detalles sobre la asociación.

### Recursos adicionales
- [Ver cómo asociar en Gestoru](https://ayuda.gestoru.com/ayuda/facturacion-pos/resolucion/gestoru)

## 2. Completa la Información de la Resolución

### Descripción breve
Este módulo detalla cómo llenar el formulario de resolución, asegurando que cada campo se complete de forma precisa para cumplir con los requisitos de la DIAN y el sistema POS.

### Características principales
- *Campos Requeridos:* Incluye campos como Nombre, Prefijo, Número inicial y final, Vigencia, Número de resolución y Mensaje corto.
- *Instrucciones Detalladas:* Cada campo cuenta con recomendaciones específicas para facilitar el proceso de llenado.
- *Soporte Multimedia:* Se sugiere el uso de imágenes y videos para ilustrar el llenado correcto del formulario.

<Card title="Campos Clave del Formulario">
  <ul>
    <li><strong>Nombre:</strong> Modalidad de la resolución (ej.: “Resolución POS 2024”).</li>
    <li><strong>Prefijo:</strong> Código asignado al crear la resolución.</li>
    <li><strong>Números:</strong> Rango autorizado (Número inicial y final).</li>
    <li><strong>Vigencia:</strong> Fechas de inicio y fin según DIAN.</li>
    <li><strong>Número de resolución:</strong> Número oficial proporcionado.</li>
    <li><strong>Mensaje corto:</strong> Breve descripción que aparecerá en las facturas.</li>
  </ul>
</Card>

<Info>
  Es crucial que la información ingresada en cada campo sea precisa y coincida con los datos oficiales registrados en la DIAN.
</Info>

### Recomendaciones generales
- Revisa cuidadosamente cada campo antes de enviar el formulario.
- Asegúrate de que la información sea actual y esté verificada.
- Utiliza los videos y capturas de pantalla disponibles para resolver dudas durante el llenado.

### Recursos adicionales
- [Cómo llenar el formulario](https://ayuda.gestoru.com/ayuda/facturacion-pos/resolucion/formulario)

## 3. Guardar y Confirmar

### Descripción breve
Este módulo cubre el proceso final para guardar la resolución en el sistema y verificar que se haya registrado correctamente en la lista de numeraciones activas.

### Características principales
- *Acción de Guardado:* Uso del botón “Guardar” para registrar la resolución.
- *Verificación Visual:* Confirmación de la correcta aparición de la numeración en el listado.
- *Soporte Multimedia:* Se recomienda el uso de imágenes y videos para validar el proceso final.

<Steps>
  <Step title="Guardar la Resolución">
    Haz clic en el botón *Guardar* para registrar la nueva numeración.
  </Step>
  <Step title="Verificar el Registro">
    Revisa que la resolución aparezca en la lista de numeraciones activas.
  </Step>
</Steps>

<Tip>
  Si gestionas varias resoluciones, marca una como <strong>Preferida</strong> para que se seleccione automáticamente al emitir facturas.
</Tip>

### Recomendaciones generales
- Asegúrate de que la información se haya guardado correctamente.
- Realiza una verificación visual para confirmar la actualización.
- Mantén un registro actualizado de las resoluciones activas para auditorías futuras.

### Recursos adicionales
- [Ver cómo guardar cambios](https://ayuda.gestoru.com/ayuda/facturacion-pos/resolucion/guardar)
