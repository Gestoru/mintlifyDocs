# Ajuste de Inventario

El submódulo **Ajuste de Inventario** permite modificar las cantidades de los productos en stock cuando existen diferencias entre el inventario físico y el registrado en el sistema.

<Warning>
Documenta siempre las razones del ajuste para facilitar futuras auditorías y evitar inconsistencias.
</Warning>

## 🚀 Características principales

- Permite ajustes manuales en el stock de productos.
- Registro detallado del motivo del ajuste.
- Opción para aumentar o disminuir las cantidades según necesidad.
- Auditoría para mantener el control de cambios.

<Steps>
  <Step title="Realizar un Ajuste de Inventario">
    1. Accede al módulo de ajustes.
    2. Selecciona el producto y la bodega donde se encuentra.
    3. Define el tipo de ajuste (incremento o reducción).
    4. Ingresa la cantidad y motivo del ajuste.
    5. Guarda los cambios.
  </Step>
</Steps>

## ✅ Recomendaciones Generales

- Realiza ajustes bajo supervisión y con respaldo documental.
- Verifica físicamente los productos antes de realizar ajustes.
- Registra los motivos para evitar confusiones futuras.
