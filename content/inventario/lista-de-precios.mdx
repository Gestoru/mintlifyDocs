# Lista de Precios

El submódulo **Lista de Precios** permite definir y gestionar diferentes listas de precios adaptadas a criterios como mayoristas o minoristas.

<Note>
La creación de listas diferenciadas puede mejorar la segmentación y atención a distintos tipos de clientes.
</Note>

## 🚀 Características principales

- Creación de listas con nombre y asignación de productos.
- Posibilidad de asignar múltiples precios a un mismo producto.

## ✅ Recomendaciones Generales

- Actualiza las listas de precios en función de cambios de mercado.
